# Meme Match Bot

Telegram-бот для подбора мемов под описанную пользователем ситуацию.

## Концепция

Пользователь описывает ситуацию текстом → бот находит семантически близкие мемы через векторный поиск → AI выбирает лучший из кандидатов → бот отправляет картинку.

## Пример использования

```
Пользователь: Когда написал код без багов с первого раза
Бот: [отправляет мем "Success Kid"]
```

## Технологический стек

| Компонент | Технология | Назначение |
|-----------|------------|------------|
| Runtime | Node.js 22+ | Среда выполнения |
| Язык | TypeScript | Типизация |
| Bot Framework | grammY | Взаимодействие с Telegram API |
| AI (выбор мема) | Groq API (Llama 3.3 70B) | Финальный выбор мема из top-5 кандидатов |
| AI (обогащение) | Groq API (Llama 3.3 70B) | Разовая генерация описаний мемов |
| Embeddings | Hugging Face Inference API | Векторизация описаний мемов и запросов |
| Мемы | Imgflip API | Источник мемов (топ-100) |

## Ключевая логика

1. **Подготовка базы (разово):** загрузка мемов → обогащение описаний через Groq → генерация эмбеддингов через Hugging Face → сохранение в `data/memes-indexed.json`
2. **При старте бота:** загрузка индексированной базы с эмбеддингами в память
3. **При запросе пользователя:**
   - Эмбеддинг запроса через Hugging Face API
   - Косинусное сходство с базой → top-5 кандидатов
   - Groq выбирает лучший мем из top-5
   - Бот отправляет картинку мема

## Требования

- Node.js 22+
- Telegram Bot Token (от @BotFather)
- Groq API Key (бесплатно на console.groq.com)
- Hugging Face API Token (бесплатно на huggingface.co)

## Быстрый старт

```bash
# Клонирование
git clone <repo>
cd meme-match-bot

# Установка зависимостей
npm install

# Настройка переменных окружения
cp .env.example .env
# Заполнить .env своими ключами

# Подготовка базы мемов (разово)
npm run fetch-memes  # Загрузка мемов из Imgflip API
npm run enrich       # Обогащение описаний через Groq
npm run embed        # Генерация эмбеддингов через HF

# Запуск
npm start
```

## Переменные окружения

```env
TELEGRAM_BOT_TOKEN=     # Токен от @BotFather
GROQ_API_KEY=           # API ключ Groq
HF_API_TOKEN=           # API токен Hugging Face
```

## Структура проекта

```

├── src/
│   ├── index.ts              # Точка входа
│   ├── bot.ts                # Инициализация бота
│   ├── config.ts             # Конфигурация
│   ├── services/

│   │   ├── meme.ts           # Сервис мемов (загрузка базы)

│   └── handlers/
│       └── message.ts        # Обработчик сообщений

├── data/

│   └── memes-indexed.json    # Описания + эмбеддинги (поисковый индекс)
├── .env.example
├── tsconfig.json
├── package.json
└── README.md
```
