# 🎭 Meme Match Bot

Telegram-бот для подбора мемов под описанную пользователем ситуацию.

## Концепция

Пользователь описывает ситуацию текстом → AI анализирует контекст и выбирает подходящий мем из базы → бот отправляет картинку.

## Пример использования

```
Пользователь: Когда написал код без багов с первого раза
Бот: [отправляет мем "Success Kid"]
```

## Технологический стек

| Компонент | Технология | Назначение |
|-----------|------------|------------|
| Runtime | Node.js 22+ | Среда выполнения |
| Язык | TypeScript | Типизация |
| Bot Framework | grammY | Взаимодействие с Telegram API |
| AI | Groq API (Llama 3.3 70B) | Анализ ситуации, выбор мема из списка |
| Мемы | Imgflip API | Источник мемов (топ-100 бесплатно) |

## Ключевая логика

1. При старте бота загружается список мемов (id + name + url) из Imgflip API
2. AI получает описание ситуации + список названий мемов
3. AI выбирает наиболее подходящий мем из списка и возвращает его ID
4. Бот отправляет картинку мема по URL


## Требования

- Node.js 22+
- Telegram Bot Token (от @BotFather)
- Groq API Key (бесплатно на console.groq.com)

## Быстрый старт

```bash
# Клонирование
git clone <repo>
cd meme-match-bot

# Установка зависимостей
npm install

# Настройка переменных окружения
cp .env.example .env
# Заполнить .env своими ключами

# Запуск
npm start
```

## Переменные окружения

```env
TELEGRAM_BOT_TOKEN=     # Токен от @BotFather
GROQ_API_KEY=           # API ключ Groq
```
